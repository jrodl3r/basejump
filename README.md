## How to use
### Install Tools
Npm, Node, Homebrew, Grunt, Ruby, Sass, Compass, Susy + Atom

### Setup Build
    $ sudo npm install -g grunt-cli
    $ npm update --save-dev
    $ vim package.json
    $ grunt

### Development
    $ grunt dev

### CSS
    $ grunt css

### JS
    $ grunt js

### Release
    $ grunt dist
